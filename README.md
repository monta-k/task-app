# タスク管理アプリ

## 概要

・Googleログイン機能

・タスク作成・編集・削除機能

・進捗はタスク編集画面で変更可能

・検索フォームに文字を入力すると、タスク名・登録者・担当者のいずれかに部分一位するタスクのみ表示される
